学习webpack的时候想着自己手动通过进行webpack配置来进行搭建一个vue项目，于是就有了这个高仿淘宝APP.<br>
<br>
项目实现：<br>
1.淘宝基本页面的ui显示;<br>
2.登录注册流程的实现;<br>
3.简单的商品购买流程;<br>
4.淘宝主题一键切换功能;<br>
<br>
开发环境：<br>
前台：vuejs、vue-router、vuex、axios、mint ui、mockjs、webpack<br>
后台：nodejs、mysql<br>
<br>
这里是包含了前台的相关代码，后台只是为了实现登录注册。<br>
